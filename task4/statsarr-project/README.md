# C++ Statistics Array Project

Практическое задание 4

## Сборка

Проект собирается через **MSYS**:

```bash
make
./statsarr.exe
```

## Задание

Класс `StatsArray` наследуется от класса `IntDynArray` (из ПЗ 2) и добавляет следующие методы:

- **average()** - вычисление среднего значения элементов массива
- **median()** - вычисление медианного значения элементов массива
- **min()** - возврат наименьшего элемента массива
- **max()** - возврат наибольшего элемента массива

## Описание методов

### double average() const
Вычисляет среднее арифметическое всех элементов массива. Бросает исключение `std::runtime_error`, если массив пуст.

### double median() const
Вычисляет медианное значение элементов массива. Для массива с нечетным количеством элементов возвращает средний элемент после сортировки. Для массива с четным количеством элементов возвращает среднее арифметическое двух средних элементов. Бросает исключение `std::runtime_error`, если массив пуст.

### int min() const
Возвращает наименьший элемент массива. Бросает исключение `std::runtime_error`, если массив пуст.

### int max() const
Возвращает наибольший элемент массива. Бросает исключение `std::runtime_error`, если массив пуст.
