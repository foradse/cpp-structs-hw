# C++ Template Array Project

Практическое задание 9

## Описание

Проект реализует шаблонный класс для массива элементов задаваемого типа с поддержкой:
- Базовых операций с элементами (get, set)
- Специального сеттера с проверкой данных для числовых типов
- Операции вывода через оператор `<<`
- Вычисления Евклидова расстояния между массивами (только для числовых типов)

## Особенности реализации

1. **Шаблонный класс `Array<T>`** - может работать с любым типом данных
2. **Метод `set_checked()`** - доступен только для числовых типов (использует SFINAE)
3. **Оператор `<<`** - для удобного вывода массива
4. **Функция `euclidean_distance()`** - вычисляет расстояние между векторами:
   - Выбрасывает `std::bad_typeid` для нечисловых типов
   - Выбрасывает `std::invalid_argument` для массивов разного размера

## Сборка

Проект собирается через **MSYS**:

```bash

./template_array.exe
```

## Использование

### Примеры:

```cpp
// Массив целых чисел
Array<int> arr1(5);
arr1.set(0, 10);
arr1.set_checked(1, 20);  // С проверкой (только для числовых типов)
std::cout << arr1 << "\n";  // [10, 20, 0, 0, 0]

// Массив вещественных чисел
Array<double> arr2(3);
arr2.set(0, 1.5);
arr2.set(1, 2.7);
std::cout << arr2 << "\n";  // [1.5, 2.7, 0]

// Вычисление Евклидова расстояния
Array<int> arr3(3);
arr3.set(0, 1);
arr3.set(1, 2);
arr3.set(2, 3);

Array<int> arr4(3);
arr4.set(0, 4);
arr4.set(1, 5);
arr4.set(2, 6);

double dist = euclidean_distance(arr3, arr4);
// dist = sqrt((1-4)^2 + (2-5)^2 + (3-6)^2) = sqrt(27) ≈ 5.196

// Для нечисловых типов - ошибка компиляции или std::bad_typeid
Array<std::string> arr5(2);
Array<std::string> arr6(2);
// double d = euclidean_distance(arr5, arr6);  // Выбросит std::bad_typeid
```
